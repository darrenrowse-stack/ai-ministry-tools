# Ministry AI Tools                                                                                                                                          
                                                                                                                                                               
  A collection of Claude Code commands I've built for ministry and sermon preparation. These are slash commands that run inside [Claude                        
  Code](https://claude.ai/code) and connect to my Obsidian vault.                                                                                              
                                                                                                                                                               
  Feel free to adapt them for your own setup.                                                                                                                  
   
  ## Commands                                                                                                                                                  
                                                         
  ### `/brew`
  A slow-thinking session that surfaces a 'grain' from your notes and partners with you to develop original thought. Either picks something at random from your
   vault or holds space for an idea you've brought. Acts as a thinking partner — one question at a time — rather than an answer machine.                       
   
  [Read more about how /brew works →](commands/brew.md)                                                                                                        
                                                         
  ## How to use these commands                                                                                                                                 
                                                         
  1. Install [Claude Code](https://claude.ai/code)
  2. Copy the command file into your `.claude/commands/` folder (create it if it doesn't exist)
  3. Run it from Claude Code with `/brew` (or whatever the command is named)                                                                                   
   
  Note: Some commands reference my Obsidian vault structure. You'll need to adapt folder paths to match your own setup.
